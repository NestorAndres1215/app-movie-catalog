# Movie Catalog

**Movie Catalog** es una aplicación web para explorar y descubrir películas, series y actores utilizando la API de [TMDb](https://www.themoviedb.org/documentation/api). Permite buscar, filtrar por géneros, ver tendencias, estrenos y gestionar favoritos de manera local.

---

## ⚡ Tecnologías y dependencias

- **Frontend**: React 18 + Vite
- **Estilos**: SCSS + Bootstrap 5
- **UI Components**: Material UI (opcional), FontAwesome
- **Rutas**: React Router v6
- **Gestión de estado**: React Context (para favoritos)
- **API**: TMDb API
- **HTTP Client**: Axios
